# Pattern Generator and Number Analyzer

A simple **Python command-line application** that allows users to
generate star patterns and analyze numbers within a given range.

## Project Description

This project is a **menu-driven Python program** with two main
utilities:

1.  **Pattern Generator** -- Generates a triangle star pattern based on
    user input.
2.  **Number Analyzer** -- Identifies whether numbers in a range are
    even or odd and calculates their total sum.

The program runs continuously until the user chooses to exit.

## Features

-   Menu-based interactive program
-   Generates triangle star patterns
-   Checks numbers as **even or odd**
-   Calculates the **sum of numbers in a range**
-   Beginner-friendly Python code

## Technologies Used

-   Python 3
-   Command Line Interface (CLI)
